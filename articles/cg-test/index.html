Introduction of Computer Graphics
Computer Graphics involves technology to access. The Process transforms and presents
information in a visual form. The role of computer graphics insensible. In today life, computer
graphics has now become a common element in user interfaces, T.V. commercial motion
pictures.
Computer Graphics is the creation of pictures with the help of a computer. The end product of
the computer graphics is a picture it may be a business graph, drawing, and engineering.
In computer graphics, two or three-dimensional pictures can be created that are used for
research. Many hardware devices algorithm has been developing for improving the speed of
picture generation with the passes of time. It includes the creation storage of models and image
of objects. These models for various fields like engineering, mathematical and so on.
Computer Graphics is the creation and manipulation of images or pictures with the help of
computers. The major product of computer graphics is a picture. With the help of CG,
pictures can be represented in 2D and 3D space. Many applications show various parts of the
displayed picture changing in size and orientation.

Key characteristics of computer graphics include:

  • Pixel-Based Structure: Most computer images (raster/bitmap) are composed of a grid of small, colored pixels.
  • Vector Representations: Alternatively, images can be drawn using mathematical formulas that define shapes and lines, allowing for infinite scaling without quality loss.
  • Rendering: The process of converting 3D model data into a 2D image, which involves calculating lighting, textures, and shading to create realistic visuals.
  • Interactive Capabilities: Users can modify images, control, and manipulate objects in real-time, such as in UI design or gaming.
  • Animation: The ability to produce moving pictures, controlling speed and detail, known as motion dynamics.
  • Modeling and Visualization: Computer graphics create, represent, and visualize both 3D synthetic objects and complex data sets.
  • Hardware Dependence: Relies on dedicated components like frame buffers, GPU processing, and display monitors.
  • Dynamic Updates: The ability to change colors, shapes, and other properties of objects, known as update dynamics. 

Key Classifications of Computer Graphics:

  • Based on Image Composition:
    • Raster Graphics (Bitmaps): Composed of a grid of pixels (e.g., photographs) and are resolution-dependent, losing quality when resized.
    • Vector Graphics: Created using mathematical formulas to define lines, curves, and shapes, allowing for scaling without quality loss.
  • Based on Interaction:
    • Interactive Computer Graphics: Involves two-way communication between the user and the computer, allowing real-time manipulation of images (e.g., video games).
    • Passive Computer Graphics (Non-interactive): Pre-computed, one-way graphics, such as movies or animations, where the user cannot change the image.
  • Based on Dimensionality:
    • 2D Graphics: Flat images that use 2D models.
    • 3D Graphics: Utilize 3D models to create realistic, rendered images with depth.
  • Based on Application:
    • Computer-Aided Design (CAD): Used in engineering/architecture for design.
    • Presentation Graphics: Used to visualize data via charts and graphs.
    • Computer Art: Used for digital paintings and graphics.
    • Scientific Visualization: Used to analyze complex scientific data.

Midpoint Circle Drawing Algorithm (MPCDA)

Given:
Radius r=14
Center (0,0)
Step 1: Initialization
Starting point:
(x0,y0)=(0,14)
Initial decision parameter:
P0=1−r=1−14=−13

Step 2: Iterative Point Calculation (First Octant)
Rules used:
• If Pk<0
Next point → (xk+1,yk)
Pk+1=Pk+2xk+1
• If Pk≥0
Next point → (xk+1,yk−1)
Pk+1=Pk+2xk−2yk+1
Iteration continues until x=y
